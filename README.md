Liso Theme
==========

Liso is an eclectic dark theme inspired by TangoTango.

**Warning:** This theme does not work well when Emacs runs in the Terminal.

### Modes

* font-lock
* mode-line
* flycheck
* helm
* anzu
* company
* diredp
* magit
* org
* ediff
* markdown
* eshell
* shell
* whitespace
* regexp-builder
* erc
* twittering
